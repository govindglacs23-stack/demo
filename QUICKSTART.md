# Quick Start Guide\n\n## 5-Minute Setup\n\n### Prerequisites\n- Node.js installed (download from nodejs.org)\n- MongoDB running (local or MongoDB Atlas account)\n\n### Step 1: Backend Setup\n\n```bash\ncd backend\nnpm install\ncp .env.example .env\n```\n\nEdit `.env`:\n```\nMONGODB_URI=mongodb://localhost:27017/lms\nJWT_SECRET=your_secret_key_here\nPORT=5000\n```\n\nStart backend:\n```bash\nnpm run dev\n```\n\nBackend running at: http://localhost:5000\n\n### Step 2: Frontend Setup (New Terminal)\n\n```bash\ncd frontend\nnpm install\nnpm run dev\n```\n\nFrontend running at: http://localhost:3000\n\n## Login\n\nCreate a test account or use demo credentials:\n- Email: test@example.com\n- Password: password\n\nThen register a new account through the app.\n\n## Key Endpoints to Test\n\n### Register User\n```\nPOST http://localhost:5000/api/auth/register\nBody: {\"name\": \"John\", \"email\": \"john@test.com\", \"password\": \"password\", \"role\": \"student\"}\n```\n\n### Login\n```\nPOST http://localhost:5000/api/auth/login\nBody: {\"email\": \"john@test.com\", \"password\": \"password\"}\n```\n\n### Get Courses\n```\nGET http://localhost:5000/api/courses\n```\n\n## Useful Commands\n\n**Backend:**\n- `npm run dev` - Start development server with nodemon\n- `npm start` - Start production server\n\n**Frontend:**\n- `npm run dev` - Start Vite dev server\n- `npm run build` - Build for production\n- `npm run preview` - Preview production build\n\n## Project Features\n\n✅ User Authentication (JWT)\n✅ Three Roles (Student, Instructor, Admin)\n✅ Course Management\n✅ Enrollment System\n✅ Progress Tracking\n✅ Quiz System\n✅ Dark Mode\n✅ Responsive Design\n✅ Search & Filter\n\n## Folder Structure Highlights\n\n```\nbackend/\n  ├── controllers/    # Business logic\n  ├── models/         # Database schemas\n  ├── routes/         # API endpoints\n  ├── middleware/     # Authentication\n  └── server.js       # Entry point\n\nfrontend/\n  ├── pages/          # Page components\n  ├── components/     # Reusable components\n  ├── services/       # API calls\n  ├── App.jsx         # Main component\n  └── main.jsx        # Entry point\n```\n\n## Troubleshooting\n\n**Port Already in Use:**\n```bash\n# Change port in backend .env\nPORT=5001\n\n# Change port in frontend vite.config.js\nport: 3001\n```\n\n**MongoDB Connection Error:**\n- Check MongoDB is running\n- For Windows: Run `mongod` in another terminal\n- For MongoDB Atlas: Check connection string and whitelist IP\n\n**CORS Error:**\n- Ensure backend is running on port 5000\n- Ensure frontend is running on port 3000\n- Check FRONTEND_URL in backend .env\n\n## Next Steps\n\n1.
