# LMS - Learning Management System\n\nA complete full-stack Learning Management System built with React, Node.js, Express, and MongoDB. This is a professional, modern web application suitable for academic projects and real-world deployment.\n\n## Features\n\n### Core Features\n- **User Authentication**: Register and login with JWT-based authentication\n- **Role-Based Access**: Three roles - Student, Instructor, and Admin\n- **Course Management**: Create, edit, and delete courses\n- **Enrollment System**: Students can enroll in courses\n- **Video Lectures**: Upload and manage course videos\n- **Progress Tracking**: Monitor student progress in courses\n- **Quiz System**: Create quizzes and track scores\n- **Search & Filter**: Find courses by category, level, and search keywords\n- **Responsive Design**: Works perfectly on desktop, tablet, and mobile\n\n### Extra Features\n- **Dark Mode Toggle**: Switch between light and dark themes\n- **Modern UI**: Clean, professional design with Tailwind CSS\n- **Real-time Stats**: Dashboards with key metrics\n- **Certificate System**: Issue certificates upon course completion\n\n## Tech Stack\n\n### Frontend\n- **React 18**: UI library with hooks\n- **React Router**: Client-side navigation\n- **Axios**: HTTP client for API calls\n- **Tailwind CSS**: Utility-first CSS framework\n- **React Icons**: Icon library\n- **Vite**: Fast build tool and dev server\n\n### Backend\n- **Node.js**: JavaScript runtime\n- **Express**: Web framework\n- **MongoDB**: NoSQL database\n- **Mongoose**: MongoDB ODM\n- **bcryptjs**: Password hashing\n- **jsonwebtoken**: JWT authentication\n- **CORS**: Cross-origin resource sharing\n\n## Project Structure\n\n```\nLMS/\n├── backend/\n│   ├── config/\n│   │   └── db.js                 # Navigate to backend directory:\n```bash\ncd backend\n```\n\n2. Install dependencies:\n```bash\nnpm install\n```\n\n3. Create `.env` file (copy from `.env.example`):\n```bash\ncp .env.example .env\n```\n\n4. Update `.env` with your MongoDB connection:\n```\nMONGODB_URI=mongodb://localhost:27017/lms\nJWT_SECRET=your_super_secret_key_change_this_in_production\nJWT_EXPIRE=7d\nPORT=5000\nNODE_ENV=development\nFRONTEND_URL=http://localhost:3000\n```\n\n5. Start MongoDB (if local):\n```bash\n# Windows\nmongod\n\n# macOS/Linux\nmongod\n```\n\n6. Start backend server:\n```bash\nnpm run dev\n```\n\nServer will run on `http://localhost:5000`\n\n### Frontend Setup\n\n1. Open new terminal and navigate to frontend directory:\n```bash\ncd frontend\n```\n\n2. Install dependencies:\n```bash\nnpm install\n```\n\n3. Start development server:\n```bash\nnpm run dev\n```\n\nApp will run on `http://localhost:3000`\n\n## API Endpoints\n\n### Authentication\n- `POST /api/auth/register` - Register new user\n- `POST /api/auth/login` - Login user\n- `GET /api/auth/me` - Get current user (protected)\n- `PUT /api/auth/update-profile` - Update profile (protected)\n\n### Courses\n- `GET /api/courses` - Get all published courses (with filters)\n- `GET /api/courses/:id` - Get single course\n- `POST /api/courses` - Create course (instructor)\n- `PUT /api/courses/:id` - Update course (instructor)\n- `DELETE /api/courses/:id` - Delete course (instructor)\n- `POST /api/courses/:id/videos` - Add video to course\n- `DELETE /api/courses/:courseId/videos/:videoId` - Delete video\n- `PUT /api/courses/:id/publish` - Publish course\n\n### Enrollments\n- `POST /api/enrollments/enroll` - Enroll in course\n- `GET /api/enrollments/my-enrollments` - Get my enrollments\n- `GET /api/enrollments/:courseId` - Get enrollment details\n- `PUT /api/enrollments/:courseId/progress` - Update progress\n- `POST /api/enrollments/:courseId/certificate` - Issue certificate\n\n### Quizzes\n- `POST /api/quizzes` - Create quiz\n- `GET /api/quizzes/course/:courseId` - Get course quizzes\n- `GET /api/quizzes/:id` - Get single quiz\n- `POST /api/quizzes/submit` - Submit quiz answers\n- `PUT /api/quizzes/:id` - Update quiz\n- `DELETE /api/quizzes/:id` - Delete quiz\n\n## Demo Credentials\n\nAfter running seed data (create in database):\n\n**Admin Account**\n- Email: `admin@example.com`\n- Password: `password123`\n\n**Instructor Account**\n- Email: `instructor@example.com`\n- Password: `password123`\n\n**Student Account**\n- Email: `student@example.com`\n- Password: `password123`\n\n## Usage Guide\n\n### For Students\n1.
