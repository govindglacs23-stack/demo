# 🚀 LMS - Quick Start for Windows\n\n## Option 1: Use the Easy Menu (Recommended)\n\nDouble-click `RUN_LMS.bat` in the LMS folder\n\nA menu will appear with options:\n1. **Install Backend** - Installs backend dependencies\n2. **Install Frontend** - Installs frontend dependencies  \n3. **Install Both** - Installs both at once\n4. **Run Backend** - Starts backend server (port 5000)\n5. **Run Frontend** - Starts frontend server (port 3000)\n6. **Exit**\n\n## Step-by-Step Setup\n\n### Step 1: Install Dependencies\n\nOption A: Use the menu (RUN_LMS.bat) and select option 3 (Install Both)\n\nOption B: Manual installation\n- Go to `backend` folder, double-click `install.bat`\n- Go to `frontend` folder, double-click `install.bat`\n\n**Wait for each to complete**\n\n### Step 2: Prerequisites\n\nBefore running, make sure you have:\n- ✅ Node.js installed (https://nodejs.org)\n- ✅ MongoDB running:\n  - Windows: Download from https://www.mongodb.com/try/download/community\n  - Or use MongoDB Atlas (cloud): https://www.mongodb.com/cloud/atlas\n  - Start MongoDB (Windows: run `mongod` in Command Prompt)\n\n### Step 3: Start the Application\n\n**Open 2 separate Command Prompt windows:**\n\nWindow 1 - Backend:\n```\ncd C:\\Users\\gopal\\OneDrive\\Desktop\\LMS\\backend\nnpm run dev\n```\n\nWindow 2 - Frontend:\n```\ncd C:\\Users\\gopal\\OneDrive\\Desktop\\LMS\\frontend\nnpm run dev\n```\n\nOr use RUN_LMS.bat:\n- Option 4: Run Backend\n- Option 5: Run Frontend (in another command prompt)\n\n### Step 4: Access the App\n\n- **Frontend**: http://localhost:3000\n- **Backend API**: http://localhost:5000/api\n\n### Step 5: Create Account & Login\n\n1. Click **\"Register\"** button\n2. Fill in:\n   - Name: Your name\n   - Email: test@example.com\n   - Password: password123\n   - Role: Student (or Instructor)\n3. Click **\"Register\"**\n4. You're logged in! Explore courses\n\n## Troubleshooting\n\n### npm command not found\n- Install Node.js from https://nodejs.org\n- Restart Command Prompt after installation\n\n### MongoDB connection error\n- Make sure MongoDB is running\n- Check `.env` file in backend folder has correct MongoDB URI\n- For local: `mongodb://localhost:27017/lms`\n- For Atlas: Copy connection string from MongoDB Atlas\n\n### Port 5000 or 3000 already in use\n- Stop other services using these ports\n- Or edit port numbers in:\n  - Backend: `backend/.env` (change PORT)\n  - Frontend: `frontend/vite.config.js` (change port)\n\n### Still having issues?\n- Check browser console (F12) for errors\n- Check Command Prompt for error messages\n- Make sure both backend and frontend are running\n- Make sure MongoDB is running\n\n## Project Structure\n\n```\nLMS/\n├── backend/          # Express.js API server\n├── frontend/         # React web application\n├── RUN_LMS.bat       # Easy menu to install & run\n├── START_HERE.md     # This file\n├── README.md         # Full documentation\n└── QUICKSTART.md     # Quick setup guide\n```\n\n## Features\n\n✅ User Registration & Login\n✅ Three Roles: Student, Instructor, Admin\n✅ Course Listings with Search & Filter\n✅ Enroll in Courses\n✅ Video Lectures\n✅ Progress Tracking\n✅ Quiz System\n✅ Dark Mode\n✅ Responsive Design\n\n## Next Steps\n\n1.
